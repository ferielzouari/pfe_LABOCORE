LABOCORE - Lab Staff Companion Frontend

This is a minimal Vite + React + TypeScript frontend scaffold for the LABOCORE lab staff companion app.

Features implemented (frontend only):
- React Router based routing
- Mock auth using React Context + localStorage
- Protected routes
- Sidebar and Topbar layout
- Pages for Dashboard, Samples, Results, Inventory, Suppliers, Technicians, Reports, Settings
- Simple CSS-only UI (no external UI libraries)

Run:

npm install
npm run dev
